# Momentum API

## Overview
The Momentum API analyzes over-by-over cricket data and converts it into an interpretable momentum view of an innings. It uses rule-based logic to classify each over as Positive, Neutral, or Negative and also identifies trends such as Increasing, Decreasing, or Stable performance.

---

## Features
- Over-wise momentum classification
- Trend detection (Increasing / Decreasing / Stable)
- Structured JSON response for dashboards
- Input validation and error handling
- Lightweight FastAPI backend

---

## Tech Stack
- Python
- FastAPI
- Uvicorn
